# PE03 – Programming Exercise (CS 132)

This program was created for **CS 132** at **City University of Seattle** and demonstrates basic object-oriented programming concepts in Java, including classes, objects, constructors, and static variables.

The program assigns a unique, auto-incremented ID to each book when it is added to the library.

### Features
- Stores books using a `Book` class
- Assigns a unique ID to each book
- Displays all books in the library
- Searches for a book by title
- Shows the library’s maximum capacity

### How to Run
Run the `App` class to view the program output in the console.
